ONLINE APPOINTMENT BOOKING SYSTEM

Project information

  • Category: Web design
  • Project date: Oct 2023
  • Language used: HTML,CSS,BOOSTRAP,PHP,MYSQL.


PROJECT ABSTRACT

Online appointment booking system that provides patients or any user a easy way of booking a doctor’s appointment online. This is a web based application that overcomes the issue of managing and booking appointments according to user choice or demands. Our software has the facility to give a unique id for every patient and store their details of every patient and the staff. A username and password are required for logging in the online appointment booking system. The system uses PHP as the front-end software which is an Scripting language and has connectivity with MYSQL as the back-end software.

ABOUT THE PROJECT

In this busy world we don’t have time to wait in infamously long in hospital queues. The problem is, queuing at clinic is often managed by administrative staff, and then wait for our turn then ask the doctor and the most frustrating thing. ONLINE APPOINTMENT BOOKING SYSTEM will help us to overcome all these problems because now patient can book their appointments at home, and then the manager assign a doctor to that particular clinic according to their appointment date.
